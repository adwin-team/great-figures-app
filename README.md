# 偉人伝学習アプリ

歴史上の偉人について楽しく学べる、ゲーミフィケーション要素を取り入れた学習Webアプリケーションです。

## 特徴

- **クイズモード**: 3つの難易度（初級・中級・上級）で偉人に関する問題に挑戦
- **偉人図鑑**: 学習した偉人の詳細情報を閲覧
- **ゲーミフィケーション**: レベルシステム、ポイント、バッジ獲得
- **学習進捗管理**: LocalStorageで進捗を自動保存
- **統計機能**: 正解率、カテゴリー別習熟度を可視化

## 使い方

1. ローカルサーバーを起動
   ```bash
   cd /home/yamane/workspace_antigravity/great-figures-app
   python3 -m http.server 8000
   ```
2. ブラウザで `http://localhost:8000/` を開く
3. ホーム画面から「クイズ開始」をクリック
4. 難易度を選択してクイズに挑戦
5. 正解してポイントを獲得し、レベルアップ
6. 偉人図鑑で学習した偉人の詳細を確認
7. 統計画面で学習進捗を確認

## ゲーミフィケーション要素

### レベルシステム
- レベル1〜50
- 必要経験値: `100 × レベル²`
- クイズ正解で経験値獲得

### ポイントシステム
- 初級: 10ポイント
- 中級: 20ポイント
- 上級: 30ポイント
- 連続正解ボーナス: 3問連続で+10、5問連続で+30

### バッジシステム
- 初心者: 初回クイズ完了
- 学習者: 累計50問正解
- 博識: 累計200問正解
- 完璧主義者: 10問全問正解
- 7日連続: 7日連続でプレイ
- 30日連続: 30日連続でプレイ
- カテゴリーマスター: 各カテゴリー全制覇
